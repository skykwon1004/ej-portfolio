Project Overview
농심몰 2주년 기념 이벤트 페이지 는 고객이 직접 참여하며 즐길 수 있는 인터랙티브 게임형 프로모션 으로 구성되었습니다.

두 가지 미니게임 — ‘농심몰 짝 맞추기(카드 짝 맞추기)’ 와 ‘환상의 짝궁(이름 궁합 테스트)’ 을 통해 사용자가 브랜드와 자연스럽게 상호작용하며 이벤트에 참여할 수 있도록 기획되었습니다.

퍼블리싱 전반을 담당했으며, 백엔드 개발자와 협의하여 로그인 상태, 쿠폰 발급 여부 등의 이벤트 조건이 원활히 반영되도록 화면을 구현했습니다.

전체 페이지는 시각적 완성도와 인터랙션의 재미를 동시에 고려해 단순하지만 참여감 있는 UX 로 구성했습니다.
Development Goal
단순한 프로모션을 넘어서, 사용자가 ‘직접 참여하는 즐거움’을 느낄 수 있는 이벤트 경험을 목표로 했습니다.
인터랙션 중심의 퍼블리싱과 부서 간 협업을 통해 브랜드 캠페인의 완성도를 높였습니다.
🃏

농심몰 짝 맞추기

사용자가 제한 시간 내에 같은 카드를 맞추는 게임으로, 시각적 재미와 몰입감을 주는 애니메이션 퍼블리싱을 구현했습니다.

💞

환상의 짝궁 테스트

이름을 입력하면 궁합 점수를 보여주는 미니게임으로, 사용자의 참여를 유도하고 이벤트 몰입도를 높였습니다.

🤝

부서 협업

기획·디자인·개발팀과의 협업을 통해 데이터 연동 구조와 이벤트 조건 로직을 함께 조율했습니다.

🎨

브랜드 톤 유지

농심몰의 브랜드 컬러와 그래픽 톤을 유지하면서, 이벤트 특유의 경쾌한 분위기를 연출했습니다.

Concept
참여형 재미
단순한 클릭 이벤트가 아닌, 사용자가 직접 결과를 만들어가는 인터랙션으로 브랜드와의 긍정적인 경험을 강화했습니다.
직관적 동선
복잡한 설명 없이도 바로 참여할 수 있는 구조로 설계하여, 사용자가 게임에 쉽게 몰입할 수 있도록 했습니다.
완성도 높은 구현
디자인 시안에 맞춰 인터랙션의 속도와 타이밍을 세밀하게 조정하여, 퍼블리싱 단계에서 시각적 완성도를 높였습니다.
PC Design
농심몰의 브랜드 톤을 유지하며, 두 가지 미니게임을 각각의 개성에 맞게 표현했습니다.
카드 짝맞추기는 시각적인 움직임을 중심으로, 환상의 짝궁 테스트는 텍스트 입력 기반의 참여 경험으로 구성했습니다.
각 섹션의 흐름을 명확히 구분하고, 사용자가 자연스럽게 이벤트를 이어갈 수 있도록 퍼블리싱 단계에서 세밀하게 조정했습니다.
