Project Overview
MBTCs는 친환경 블록체인 기반의 메타버스 생태계 를 소개하기 위한 프로모션 웹사이트입니다.

복잡한 기술적 개념을 시각적으로 명확하게 전달하고, 투자자와 사용자에게 브랜드의 비전과 신뢰감 을 동시에 주는 것을 목표로 했습니다.

메인 비주얼은 ‘Eco-friendly Blockchain Metaverse’ 메시지를 중심으로 3D 그래픽과 네온 라이트 효과 를 활용하여 미래지향적이고 기술적인 브랜드 이미지 를 구현했습니다.

또한 다크 모드 기반의 레이아웃과 그린 포인트 컬러 를 사용해 친환경 에너지와 디지털 혁신이라는 브랜드 아이덴티티를 시각적으로 표현했습니다.
Design Goal
사용자 중심의 정보 전달과 기술적 신뢰감의 균형을 목표로,
브랜드 비전과 프로젝트의 기술적 메시지를 직관적으로 보여주는 디자인을 구현했습니다.
