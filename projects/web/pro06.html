Project Overview
PPO#은 피부과 병원이 자체 제작한 화장품을 내원 고객에게 소개하고 판매하는 병원 전용 커머스 웹사이트입니다.

외부 결제 시스템 없이, 고객이 상품을 장바구니에 담은 뒤 데스크에 화면을 보여주면 병원에서 직접 결제가 이루어지는 비결제형 쇼핑 UX를 가집니다. PC와 모바일 환경 모두를 지원하는 반응형으로 제작했으며, 한국어·영어·중국어·일본어 등 6개 언어를 지원해 외국인 내원 고객까지 타겟으로 합니다.
UX 설계 포인트
일반 쇼핑몰과 다른 운영 구조에서,
사용자가 자연스럽게 구매 흐름을 마칠 수 있도록 설계했습니다.
비결제 UX 흐름

결제 버튼 대신 데스크 안내

일반 쇼핑몰의 "결제하기" CTA 대신, 장바구니 완료 시 "데스크에 이 화면을 보여주세요" 안내 팝업으로 구매 흐름을 마무리하도록 설계했습니다. 결제 기능이 없는 상황에서 사용자 혼란을 줄이는 핵심 결정이었습니다.

컬렉션 페이지

엣지 케이스까지 설계

그리드/리스트 레이아웃 토글, 카테고리 필터, 품절(SOLD OUT) 상태 처리, 빈 카테고리 화면, 404 페이지까지 설계해 실제 운영 환경을 고려한 완성도를 높였습니다.

다국어 지원

6개 언어 전환 UI

KOR · ENG · CHN · TH · JPN · RU 언어 전환 UI를 설계했습니다. 외국인 내원 고객도 불편 없이 탐색할 수 있도록 언어별 레이아웃 깨짐을 최소화했습니다.

Design Goal
브랜드 무드와 제품 탐색 경험을 함께 고려해,
명확하고 신뢰감 있는 코스메틱 웹 경험을 설계했습니다.
💡

Clarity

제품 정보와 핵심 메시지를 직관적으로 전달할 수 있도록 정보 구조를 정리했습니다.

🎨

Consistency

브랜드 무드에 맞는 컬러와 UI 요소를 일관되게 유지해 전체 인상을 정돈했습니다.

🧴

Brand Mood

깨끗하고 부드러운 코스메틱 무드를 통해 제품과 브랜드의 신뢰감을 함께 표현했습니다.

🧭

Usability

옵션 선택과 장바구니 흐름을 고려해 사용자가 자연스럽게 탐색할 수 있도록 설계했습니다.

📱

Responsive

PC와 모바일 환경 모두에서 브랜드 경험과 가독성이 유지되도록 반응형으로 구성했습니다.

Concept
Clean
군더더기 없는 레이아웃과 여백 중심의 구조를 통해 제품과 핵심 정보가 깔끔하게 드러나도록 디자인했습니다.
Trustworthy
병원 판매용 화장품이라는 특성을 반영해 안정감 있는 정보 배치와 정돈된 UI로 브랜드의 신뢰감을 강화했습니다.
Soft Cosmetic Mood
코스메틱 브랜드 특유의 부드러운 분위기를 담아 제품의 매력과 브랜드 감성을 자연스럽게 전달하도록 구성했습니다.
Design System
Tailwind CSS 디자인 시스템을 Figma에 직접 연동해
개발자가 코드로 바로 옮길 수 있는 수준으로 토큰을 정의했습니다.
개발 친화적 핸드오프

Figma에 Tailwind CSS 공식 디자인 시스템 플러그인을 설치하고, Tailwind config 기준에 맞춰 컬러·타이포·여백·브레이크포인트 토큰을 정의했습니다. 브랜드 전용 컬러는 커스텀 값으로 별도 지정했으며, 개발자에게 Tailwind 클래스명을 직접 명시한 스펙과 환경 설정 방법을 함께 제안했습니다. 디자이너가 개발 흐름을 이해하고 있을 때 협업 비용이 얼마나 줄어드는지를 직접 실천한 경험이었습니다.

잘한 점

Tailwind CSS 기반 디자인 시스템을 구축하고 개발자에게 클래스명과 환경 설정 방법까지 제안했습니다. 2주라는 짧은 기간 안에 PC·모바일 전 화면을 완성하면서, 디자이너가 개발 흐름을 이해할 때 협업 비용이 어떻게 줄어드는지를 직접 경험했습니다.

아쉬운 점 & 다음엔

병원이 직접 만든 화장품이라는 차별점을 콘텐츠 UX로 풀어내지 못했습니다. 성분 신뢰도, 시술 후 홈케어 연계 등을 정보 구조에 녹였다면 브랜드 설득력이 더 높아졌을 것입니다. 다음 프로젝트에서는 기획 단계부터 브랜드 스토리를 UX 흐름 안에 포함시키는 방향으로 제안해보고 싶습니다.
